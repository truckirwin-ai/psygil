Practice Admin

For supervisors and practice managers running Psygil for a group.

User roster

Settings > Practice > Users lets you add clinicians, psychometrists, admins, and front-desk staff. Each user gets a role, which controls what they can do.

Roles

  • Psychologist, full clinical workflow, diagnostic authority.
  • Psychometrist, test administration and scoring.
  • Admin, practice configuration, user roster, audit review.
  • Receptionist, intake only.

Audit log review

Settings > Audit surfaces the last N events with filters by clinician, case, and action type. The audit log is write-only. Do not try to edit it; you cannot, and your attempt is itself audited.

Shared workspace recommendations

  • Use an SMB share on a server you control, not a consumer cloud drive.
  • Ensure only practice members have read or write access at the filesystem level.
  • Back up nightly; test restores quarterly.
  • Document your share path in your HIPAA risk analysis.

Concurrency

Two clinicians cannot edit the same case simultaneously. Psygil writes a lock when a case is opened for editing; a second user sees read-only with a warning banner.
