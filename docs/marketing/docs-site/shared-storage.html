Shared Storage

Where your cases live. The workspace folder is the single source of truth for Psygil.

Supported storage

  • Local disk, fully supported. Fastest and simplest.
  • SMB or NFS network share, fully supported. Required for Practice edition.
  • Dropbox, OneDrive, Google Drive, iCloud Drive, discouraged for multi-user use. File sync can race with live editing.
Cloud drives for teams. For Practice edition with more than one clinician, use a proper SMB share (synology, truenas, a Windows Server file share) rather than a consumer cloud drive. Psygil will warn you if it detects a cloud-synced folder.

Choosing a path

  1. Open Setup Wizard > Storage, or later Settings > Workspace.
  2. Click Choose Folder.
  3. Review the validation summary: writable, free space, cloud-drive flag.
  4. Click Provision. Psygil creates the folder layout and stub templates.

Moving a workspace

Quit Psygil on every workstation that uses the workspace. Copy the entire workspace folder to the new location. Open Psygil, go to Settings > Workspace > Change, and select the new path. Psygil will re-sync the database from the filesystem.

Backups

Back up the workspace folder with your normal backup tool (Time Machine, Windows Backup, Veeam). The encrypted database lives inside the workspace, so one backup covers both files and metadata.
