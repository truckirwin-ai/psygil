Getting Started

Download, install, and launch Psygil for the first time.

System requirements

  • macOS 13 (Ventura) or newer, Apple Silicon or Intel.
  • Windows 10 or 11, 64-bit.
  • 4 GB free RAM, 2 GB free disk for the app plus workspace for cases.
  • Any word processor that opens .docx (Word, Pages, LibreOffice).

Download and install

  1. Visit psygil.com/download and choose your platform.
  2. Verify the installer is signed. On macOS, right-click the DMG and choose Open; on Windows, confirm the publisher is Foundry SMB.
  3. Run the installer. macOS: drag Psygil to Applications. Windows: accept the default install path.
  4. Launch Psygil. The first-launch Setup Wizard will appear.
Warning. Do not store case data in a browser Downloads folder or Desktop. Use the Setup Wizard to choose a proper workspace folder in Documents or on a shared drive.

What happens on first launch

Psygil creates an encrypted local database under your user profile, registers the psygil:// protocol for secure login, and opens the Setup Wizard. Nothing touches your cases until you complete the wizard and choose a workspace folder.

Next steps

  • Run the Setup Wizard
  • Choose a workspace, local or shared
