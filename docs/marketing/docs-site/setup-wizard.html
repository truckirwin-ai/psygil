Setup Wizard

Configure your edition, storage location, practice info, and appearance, one time, on first launch.

Wizard stages

  1. Welcome. Read the HIPAA notice and accept the EULA.
  2. License. Paste your license key or start a trial.
  3. Edition. Choose Individual, Practice, or Enterprise.
  4. Storage. Pick the workspace folder (see Shared Storage).
  5. Practice info. Enter your practice name, NPI, address, and contact.
  6. Clinical defaults. Select default evaluation types and your preferred test batteries.
  7. Appearance. Light or dark theme, font size.
  8. AI config. Toggle drafting assistants on or off; set tone (forensic, neutral, plain).
  9. Finish. The main workspace opens.

Choosing an edition

Individual

A single clinician on one workstation. Cases live locally. Best for solo practices.

Practice

A small group, typically 2 to 15 clinicians, sharing a workspace folder on a network drive or SMB share. Each clinician has a user record and audit trail.

Enterprise

Forthcoming. Adds SSO (SAML), centralized admin console, and cross-site audit review.

Can I change editions later? Yes. Practice admins can upgrade from Individual by reopening Setup from Settings > Practice.

What the wizard writes

  • A setup config file (psygil-setup.json) under your user profile.
  • An encrypted SQLite database at the chosen workspace root.
  • Empty cases/, reports/, and _inbox/ folders inside the workspace.
