Troubleshooting

App will not launch after install

  • macOS, System Settings > Privacy and Security: allow apps from identified developers.
  • Windows, confirm the publisher is Foundry SMB, then accept the SmartScreen prompt.

Cannot find workspace after move

Settings > Workspace > Change, then point to the new path. Psygil re-syncs from the filesystem.

A case is missing from the tree but the folder exists

Confirm the folder name matches the pattern YYYY-NNNN LastName, FirstName. If not, rename it. Psygil re-adds it on next sync.

Changes I made outside the app did not appear

File watcher debounces events. Wait 1 to 2 seconds. If still missing, Settings > Workspace > Rescan.

Gate will not advance

The gate chip tells you what is missing. For Diagnostics, every Final Formulation section must be Approved (blue chip). Editing a section revokes its approval.

Report is empty or shows placeholders

That is by design. Sections populate only after their stage is complete. Advance the pipeline, then re-build.

Export to PDF fails

  • Confirm the chosen path is writable.
  • Disable third-party PDF printers temporarily; Psygil uses its own renderer.

Crash report

Help > Save Diagnostics Bundle produces a zip with logs and non-PHI environment details. Send it to support@psygil.com.
