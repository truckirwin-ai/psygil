FAQ

Does Psygil diagnose for me?

No. Psygil helps you organize, draft, and audit. Every diagnostic decision is yours.

Does Psygil work offline?

Yes. The core workflow is fully offline. Optional AI assistants can be turned off.

Can I use Dropbox or OneDrive for the workspace?

For a single clinician on one machine, yes, with caveats. For a team, use a real SMB share. See Shared Storage.

What happens if two clinicians open the same case?

The first opens it read-write. The second sees a read-only view with a clear notice until the first closes the case.

Can I edit the report in Word after exporting?

Yes, drafts are standard .docx. Final reports are locked after Publish; reopen a draft cycle to revise.

How is the report kept from getting ahead of my clinical work?

Each section of the report is gated to the stage that produced it. Sections remain placeholders until the required stage is complete and, for Diagnostics, approved.

What file formats can I import as collateral?

PDF, DOCX, TXT, RTF, common image formats, and audio or video for attached transcripts. Drop them into the appropriate case subfolder.

Is there a mobile app?

Not today. Psygil is a desktop workspace by design. A mobile intake companion is on the long-term roadmap.
