Psygil Documentation

A practical, IDE-grade workspace for forensic psychology. The doctor makes every diagnostic decision. Psygil handles the paperwork.

Where to start

  • Install Psygil on Mac or Windows
  • Run the first-launch setup wizard
  • Create your first case
  • Export a report to Word or PDF

How Psygil is organized

Each case progresses through a six-stage pipeline: Onboarding, Testing, Interview, Diagnostics, Review, and Complete. You can only advance a stage when the work for that stage is captured and, for Diagnostics, clinically approved. The report fills in progressively as stages complete, so no section is ever ahead of your clinical judgment.

Non-negotiable principle. Psygil does not diagnose. Every diagnostic decision in the audit trail is stamped with a clinician's signature. The assistants draft, summarize, and organize, nothing more.

Editions

  • Individual, a single clinician on one workstation.
  • Practice, a small group with a shared workspace on a network drive.
  • Enterprise, forthcoming, with SSO and centralized admin.
